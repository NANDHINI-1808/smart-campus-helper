# Smart Campus Helper v2

🎓 Smart Campus Helper v2 is a college web app to manage campus life.

## Features
- Complaints: Submit & speak complaints
- Lost & Found: Post lost items
- Inter-College Events: Save & view events
- Intra-College Events: Save & view events
- Notes Maker: Save & speak notes
- Chat with Friends: Simple chat simulation

## Login
- Login with email/password or Google/GitHub/Facebook buttons (simulated)
- After login → Menu → Select a feature

## How to Run Locally
1. Download the repository
2. Open `index.html` in a browser
