# Smart Campus Helper v2

🎓 **Smart Campus Helper v2** is a web app designed for college students to manage campus life efficiently.

## Features

- 📢 Complaints: Submit complaints & let the app read them aloud.
- 🔍 Lost & Found: Post lost items and view them.
- 🌐 Inter-College Events: Save & view inter-college events with dates and locations.
- 🏫 Intra-College Events: Save & view intra-college events.
- 📝 Notes Maker: Write, save, and speak your notes.
- 💬 Chat with Friends: Simple chat simulation with speech output.

## Login

- Login with email/password or social login buttons (Google, GitHub, Facebook – simulated).
- After login, you can select features from the menu.


## How to Run Locally

1. Clone or download the repository.  
2. Open `index.html` in a browser.  
3. Start using the app.

## Technologies Used

- HTML, CSS, JavaScript  
- Local Storage for saving notes/events  
- Browser Speech Synthesis API for speaking text

## Future Improvements
